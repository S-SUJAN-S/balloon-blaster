# 🎈 Balloon Blaster

A fun browser-based balloon popping game — built entirely with **HTML, CSS & JavaScript** using vibe coding (AI-assisted development). No frameworks, no installs — just open and play!

🎮 **Live Demo → [balloon-blaster.netlify.app](https://balloon-blaster.netlify.app)**

## 🚀 Features

- 🎯 Click/tap balloons to pop them before they escape
- ⏱️ Timer-based gameplay with increasing difficulty
- 💥 Satisfying pop animations
- 📱 Works on desktop and mobile browsers
- ⚡ Zero dependencies — pure HTML/CSS/JS

---

## 🕹️ How to Play

1. Open the game in any browser
2. Click/tap the balloons as fast as you can
3. Don't let them escape — every missed balloon costs you!
4. Beat your high score 🏆
